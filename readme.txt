Add Whisper to the available oEmbed providers.

== Description ==

Adds oEmbed support for Whisper in posts. Just copy a Whisper's url from http://whisper.sh into your post and the Whisper embed will be created.

== Installation ==

1. Go to Dashboard >> Plugins >> Add New >> Search
2. Search for "Whisper oEmbed," click "Install Now," then "Activate Plugin"

== Frequently Asked Questions ==

= How do I embed a Whisper? =

Copy the Whisper url into a post and it is created automatically when viewing.
